# ☕ 스타벅스 음료 추천 웹앱

### 사용자의 주문 및 리뷰 데이터를 기반으로 개인 맞춤형 음료를 추천하는 딥러닝 기반 웹 애플리케이션

---

## 📌 앱 개요
이 애플리케이션은 스타벅스의 영양 정보 데이터를 바탕으로 사용자의 취향과 주문 이력을 분석하고, 맞춤형 음료를 추천합니다. 또한 리뷰 기반 감정 분석을 통해 추천 알고리즘을 고도화하며, 음료의 평점, 주문 수, 성분 정보 등을 시각화하여 소비자 인사이트를 제공합니다.  
<br>
애플리케이션 링크 : [Streamlit Application](https://recom-starbucks-codekookiz.streamlit.app/)

## ☕ 주요 기능

### 📦 1. 음료 추천 시스템 (5가지 모드)
사용자의 주문 내역과 취향을 분석하여 5가지 맞춤형 추천 모드를 제공합니다.
1. **늘 먹던 걸로 (Always Similar)**: 사용자가 자주 주문하는 음료와 유사한 스타일을 추천합니다.
2. **피곤해요.. 카페인 충전 (High Caffeine)**: 카페인 함량이 높은 음료를 추천하여 활력을 더해줍니다.
3. **달달한 거 주세요 (Sweet)**: 당류 함량이 높은 달콤한 음료를 추천합니다.
4. **죄책감은 최소로 (Low Calorie)**: 칼로리가 낮은 가벼운 음료를 제안합니다.
5. **새로운 시도 (New Challenge)**: 한 번도 주문해보지 않는 메뉴 중, 예상 별점이 높은 음료를 추천합니다.

### 💬 2. 리뷰 분석 및 추천 점수 반영
- **AI 감정 분석**: 사용자가 작성한 리뷰를 영문으로 번역(`opus-mt-ko-en`) 후, `DistilBERT` 모델을 통해 긍정/부정 감정을 분석합니다.
- **실시간 추천 점수**: 긍정 리뷰는 추천 점수를 높이고, 부정 리뷰는 낮추어 다음 추천에 즉각 반영됩니다.

### 📊 3. 통계 및 데이터 시각화
- **인기 차트**: 주문 수 상위/하위, 평점 상위/하위 음료를 한눈에 볼 수 있습니다.
- **상관관계 분석**: 평점과 주문 수의 상관관계를 산점도로 시각화하여 소비자 패턴을 분석합니다.
- **데이터 투명성**: 메뉴 정보, 주문 내역, 리뷰 데이터를 테이블 형태로 실시간 제공합니다.

## 🚀 로컬 설치 및 실행 (Local Installation)

이 프로젝트를 로컬 환경에서 실행하려면 다음 단계를 따르세요.

1. **저장소 클론**
   ```bash
   git clone https://github.com/codekookiz/recom-starbucks-app.git
   cd recom-starbucks-app
   ```

2. **라이브러리 설치**
   ```bash
   pip install -r requirements.txt
   ```

3. **앱 실행**
   ```bash
   streamlit run app.py
   ```
